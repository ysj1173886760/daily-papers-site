关于 Daily AI Papers

自动化的AI论文收集与分析系统

项目介绍

Daily AI Papers 是一个基于大语言模型的智能论文分析系统，专注于AI前沿研究领域，特别是检索增强生成(RAG)和知识图谱(KG)等方向。

核心功能

  • 自动获取：每日从arXiv自动获取最新论文
  • 智能过滤：使用LLM过滤相关性高的优质论文
  • 深度分析：提供结构化的论文分析，包括技术方案、创新点、实验结果等
  • RSS发布：支持RSS订阅，兼容所有主流阅读器
  • 多端适配：响应式设计，支持移动端和桌面端

技术栈

后端处理

  • Python + PocketFlow框架
  • 大语言模型分析
  • arXiv API集成

前端展示

  • 静态站点生成
  • Jinja2模板引擎
  • 响应式CSS设计

部署发布

  • GitHub Pages
  • GitHub Actions CI/CD
  • RSS 2.0标准
